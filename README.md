## 🔐 Android JWT Authentication App

**Role:** Android Developer  
**Repository:** `android_dp58_nguyenvuphuong`

An Android authentication application that handles login, token storage, protected API calls, access token refresh, and force logout when the refresh token expires.

### Highlights

- Login screen built with XML Layout
- Email/password validation
- Login API integration using Retrofit
- Store Access Token, Refresh Token, and user information using SharedPreferences
- Automatically attach `Authorization: Bearer <token>` with OkHttp Interceptor
- Call protected APIs after successful authentication
- Refresh Access Token when it expires
- Retry the original failed request after refreshing token
- Force logout when Refresh Token expires
- Display protected API data after authentication
- Git workflow with commit, push, and pull

### Flow

```text
Login
→ Save Token
→ Call Protected API
→ Refresh Access Token when expired
→ Retry Original Request
→ Force Logout when Refresh Token expires
```

### Tech Stack

`Java` `Retrofit` `OkHttp` `SharedPreferences` `REST API` `JWT Authentication`

### Source Code

[Android JWT Authentication App](https://github.com/PhuongL1/android_dp58_nguyenvuphuong)
